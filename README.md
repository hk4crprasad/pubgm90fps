### REQUIREMENTS :

* internet

* Termux

* Pubg Mobile Installed

### Installation in Termux

* `apt-get update -y`

* `apt-get upgrade -y`

* `pkg install git -y`

* `git clone https://github.com/hk4crprasad/pubgm90fps.git`

* `cd pubgm90fps`

* `chmod 700 install`

* `./install`
